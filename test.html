Screen D — 数据模型设计
Outline（大纲集合）
model Outline {
  id          String    @id @default(cuid())
  title       String                          // 大纲标题（可按卷/弧划分）
  novelId     String
  novel       Novel     @relation(fields: [novelId], references: [id], onDelete: Cascade)
  order       Int       @default(0)          // 排序序号
  items       OutlineItem[]

  createdAt   DateTime  @default(now())
  updatedAt   DateTime  @updatedAt

  @@index([novelId, order])
}
OutlineItem（大纲节点）— 核心
model OutlineItem {
  id              String    @id @default(cuid())
  outlineId       String
  outline         Outline   @relation(fields: [outlineId], references: [id], onDelete: Cascade)

  // ── 基础信息 ──
  title           String                          // 节点标题（如"第一章：拜入宗门"）
  summary         String?                         // 内容摘要
  order           Int       @default(0)          // 排序
  plotType        String?                         // scene / event / turning_point / climax

  // ── 结构化元信息（AI 提取 + 手动编辑）──
  location        String?                         // 📍 地点
  timePeriod      String?                         // 🕐 时间/季节描述

  // ── 情感变化（主角）──
  emotionBefore   String?                         // 情感状态 - 前
  emotionAfter    String?                         // 情感状态 - 后
  emotionNote     String?                         // 变化原因/说明

  // ── 关联 ──
  chapterId       String?                         // 可选关联到章节（双向）
  chapter         Chapter?  @relation("OutlineItem_Chapter", fields: [chapterId], references: [id])
  characters      Character[]                        // 涉及人物（多对多）

  createdAt       DateTime  @default(now())
  updatedAt       DateTime  @updatedAt

  @@index([outlineId, order])
  @@index([chapterId])
}
关联关系说明
// Novel ← Outline (一对多：一个作品可以有多个大纲集合)
// Outline ← OutlineItem (一对多：一个大纲包含多个节点)
// OutlineItem ↔ Character (多对多：节点涉及多个人物)
// OutlineItem → Chapter (多对一可选：节点可关联一个章节)
// Chapter → OutlineItem (一对多可选：章节可对应多个大纲节点)

// ── 需要在现有 model 中添加 ──
// Novel.model:  outlines    Outline[]
// Chapter.model: outlineItems OutlineItem[]  @relation("OutlineItem_Chapter")
// Character.model: outlineItems OutlineItem[] (多对多中间表)
