# 表单配置化生成指南

## 核心思路

通过定义配置数组 `CONFIG_FIELDS`，使用 `renderField` 函数动态渲染表单，实现 **单一数据源**。

## 步骤

### 1. 定义配置数组

创建 `src/lib/configs/xxx-defs.ts`：

```typescript
import { buildDefaultValues, type ConfigOf, type ConfigOption } from "@/lib/configs/config-utils"
import type { ConfigFieldDef } from "@/lib/configs/config-utils"

// 选项定义（icon 为 lucide 组件名，color 可选）
const statusOptions: ConfigOption[] = [
  { value: "连载中", icon: "Zap", color: "primary" },
  { value: "已完结", icon: "CircleCheck", color: "success" },
]

// 配置字段定义
export const XXX_CONFIG_FIELDS = [
  {
    key: "status",        // 字段 key，对应 config 对象的 key
    label: "状态",         // 显示标签
    section: "基本信息",   // 分组名称
    type: "single",       // single | multi | toggle
    options: statusOptions,
  },
  {
    key: "enabled",
    label: "启用",
    section: "基本信息",
    type: "toggle",
  },
] as const satisfies readonly ConfigFieldDef<string>[]

// 类型推导
export type XxxConfig = ConfigOf<typeof XXX_CONFIG_FIELDS>

// 默认值
export const DEFAULT_XXX_CONFIG = buildDefaultValues(XXX_CONFIG_FIELDS)
```

### 2. 使用 renderField 渲染

```typescript
import { renderField } from "@/lib/configs/render-utils"
import { XXX_CONFIG_FIELDS, DEFAULT_XXX_CONFIG } from "@/lib/configs/xxx-defs"

// 状态管理
const [config, setConfig] = useState<XxxConfig>({
  ...DEFAULT_XXX_CONFIG,
  status: "",
})

// 渲染表单
{XXX_CONFIG_FIELDS.map((field) =>
  renderField(field, config, setConfig)
)}
```

### 3. 保存/提交

```typescript
// 序列化为 JSON 字符串存入数据库
const configJson = JSON.stringify(config)

// 从 JSON 字符串解析
const parsed = JSON.parse(dbRecord.config)
setConfig({ ...DEFAULT_XXX_CONFIG, ...parsed })
```

## 字段类型说明

| type     | 值类型        | UI 组件              | options 要求    |
| -------- | ------------- | -------------------- | --------------- |
| `single` | `string`      | RadioGroup           | 必需            |
| `multi`  | `string[]`    | MultiRadioGroup      | 必需            |
| `toggle` | `boolean`     | Toggle               | 不需要          |

## 选项格式

```typescript
{
  value: string        // 必填，唯一标识
  icon?: string        // 可选，lucide 组件名（如 "Heart"）
  color?: OptionColor  // 可选，primary/success/warn/default
  meta?: string        // 可选，AI 可读的描述
}
```

## 分组布局

多个字段可按 `section` 分组，使用 `Card` 分别渲染：

```typescript
const sections = Array.from(new Set(FIELDS.map((f) => f.section)))
sections.map((sec) => {
  const fields = FIELDS.filter((f) => f.section === sec)
  return <Card key={sec}><CardContent>{fields.map(f => renderField(f, config, onChange))}</CardContent></Card>
})
```
