# 小说创作辅助系统 - 项目概述

## 项目定位
一款面向小说创作者的本地化辅助写作工具，提供从构思、写作到润色的全流程支持。

## 核心价值
- 🖊️ **沉浸式写作**：简洁专注的编辑器体验
- 🤖 **AI 智能辅助**：润色、灵感生成、一致性检查
- 📊 **全局掌控**：人物关系、时间线、情节脉络可视化
- 🔒 **数据安全**：所有数据本地存储，无需联网

## 目标用户
- 网络小说作者
- 同人创作者
- 需要结构化写作工具的作者
