# 小说创作辅助系统 - 项目概述

## 项目定位

一款面向小说创作者的本地化辅助写作工具，提供从构思、大纲、写作到润色的全流程支持。

## 核心价值

- 🖊️ **沉浸式写作**：简洁专注的编辑器体验
- 🤖 **AI 智能辅助**：润色、续写、扩写、头脑风暴
- 📊 **全局掌控**：人物、大纲、情节、情感追踪可视化
- 🔒 **数据安全**：所有数据本地 SQLite 存储，无需联网

## 目标用户

- **网络小说作者**：日更 4000+ 字，需要 AI 辅助
- **同人创作者**：多角色、多分支剧情的结构化需求
- **长篇创作者**：50 万字以上，需要全局一致性保障

## 项目里程碑

### 第一阶段：基础功能（已完成 ✅）

- 项目骨架搭建（Next.js + Tailwind + Prisma + SQLite）
- 作品管理（创建/删除/概览，题材多选，状态管理）
- 章节编辑器（Textarea，3s 防抖自动保存，字数统计）
- 人物管理（结构化特征：叙事功能/动机/能力/关系/弧光）
- AI 润色（规则 CRUD + 配置化编辑器 + 选中文本润色）

### 第二阶段：核心功能完善（开发中 🔄）

- 故事大纲、人物情感追踪、头脑风暴、AI 续写、AI 扩写

### 第三阶段：Agent 化与智能化（远期 🔮）

- 多 Agent 协作、事实核查、RAG 知识库、版本管理

## 项目约束

- **本地优先**：所有数据存储于本地 SQLite，无云端同步
- **模块化**：功能模块独立，可独立开发和测试
- **渐进增强**：基础功能立即可用，高级功能按需启用
