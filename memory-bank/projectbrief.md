# 小说创作辅助系统 - 项目概述

## 项目定位
一款面向小说创作者的本地化辅助写作工具，提供从构思、大纲、写作到润色的全流程支持。

## 核心价值
- 🖊️ **沉浸式写作**：简洁专注的编辑器体验
- 🤖 **AI 智能辅助**：润色、续写、扩写、头脑风暴灵感生成
- 📊 **全局掌控**：人物关系、故事大纲、情节脉络、情感追踪可视化
- 🔒 **数据安全**：所有数据本地存储，无需联网

## 目标用户
- 网络小说作者
- 同人创作者
- 需要结构化写作工具的作者

## 项目阶段

### 第一阶段（已完成）：基础功能
- [x] 项目骨架搭建（Next.js + Tailwind + Prisma）
- [x] 作品管理（CRUD + 题材多选 + 状态标签）
- [x] 章节编辑器（自动保存、状态管理、字数统计）
- [x] 人物管理（结构化特征数据：基本信息/叙事功能/能力/关系/成长弧光）
- [x] AI 润色功能（规则管理 + 编辑器内润色）
- [x] 作品概览（基本信息编辑 + 预览双模式）
- [x] 本地 SQLite 存储

### 第二阶段（开发中）：核心功能完善
- [ ] **故事大纲**：层级结构大纲，支持卷/章/情节/场景四级
- [ ] **人物情感追踪**：场景级人物情绪标记与变化曲线
- [ ] **头脑风暴**：灵感卡片 + AI 点子生成
- [ ] **AI 续写**：基于上下文自动续写
- [ ] **AI 扩写**：选中内容智能扩写
- [ ] 时间线管理
- [ ] 关系图谱可视化
- [ ] 伏笔系统

- [ ] 多 Agent 协作（规划 + 写作 + 审稿）
