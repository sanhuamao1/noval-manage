# 小说创作辅助系统

## 定位

面向小说创作者的本地化辅助写作工具，覆盖构思、大纲、写作到润色全流程。

## 核心价值

- **沉浸式写作**：简洁专注的编辑器
- **AI 辅助**：润色、续写、扩写、头脑风暴
- **全局掌控**：人物、大纲、情节、情感追踪
- **本地优先**：SQLite 存储，无云端依赖

## 里程碑

| 阶段 | 状态 | 内容 |
|------|------|------|
| 一：基础功能 | ✅ | 作品管理、章节编辑器、人物管理、AI 润色 |
| 二：核心完善 | 🔄 | 故事大纲、情感追踪、头脑风暴、AI 续写/扩写 |
| 三：Agent 化 | 🔮 | 多 Agent 协作、事实核查、RAG 知识库 |
