# 产品上下文

## 解决的核心问题

1. 信息碎片化 —— 人物/情节/设定分散
2. 一致性难维护 —— 长篇前后矛盾
3. 写作效率低 —— 缺乏专注环境 + AI 辅助
4. AI 无定制 —— 通用工具不懂创作，规则不可配置

## 使用场景

1. 日常写作：章节选择 → 编辑器（3s 防抖保存，字数统计，Tab 缩进）
2. 人物管理：点击人物 → 右侧抽屉编辑（2s 防抖保存）
3. AI 润色：选中文本 → 选择规则/样本 → AI 结果 → 确认/取消
4. 作品概览：预览 → 编辑 → 保存

## 设计原则

- 本地优先，AI 调用外无网络依赖
- 简洁高效，深色琥珀主题
- 渐进增强，基础立即可用
- Prisma 唯一数据源，级联删除

## 功能优先级

| 功能 | 优先级 |
|------|--------|
| 章节编辑器、AI 润色、人物管理、作品概览 | P0 |
| 故事大纲、AI 续写/扩写 | P1 |
| 情感追踪、头脑风暴 | P2 |
| 关系图谱、时间线、导出 | P3 |
